# Club Management Frontend (Vue2)

## Tech Stack
- Vue 2 + Vue Router + Vuex
- Element UI (primary: #4167b1, background: #def0f9)
- Axios
- Vue CLI 4
- ECharts (数据可视化)
- Sass (样式预处理)

## Setup
```bash
cd frontend
npm install
npm run serve
```

- Dev server: http://localhost:5174
- Proxy to backend: http://localhost:8081/api
- Env vars: see .env.development
